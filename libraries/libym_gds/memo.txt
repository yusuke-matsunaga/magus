								松永　裕介

- GDS-II のデータは record のリストで構成される．

- record は record type と data type を持つ．

- 簡単な属性は一つの record で表されるが，複雑な属性は複数の record の
  組み合わせで表される．

- すべての record の最初の2バイトは record 全体のバイト数

- 続く2バイトは record type と data type を表す数字
